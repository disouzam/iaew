# IAEW 2024 5K4

## Installing guide

### Crear un entorno virtual  

python -m venv venv

### En macOS/Linux

- source venv/bin/activate
- pip install fastapi
- install python libraries
- python version 3.10 o mayor
- install sqlite

### Start HTT server

- fastapi dev main.py
- HTTP server en localhost, port 8000
- http://localhost:8000/docs
